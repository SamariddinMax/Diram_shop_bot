🛍️ Diram Shop Bot

Diram Shop Bot — ин як боти мағозаи Telegram мебошад, ки бо истифода аз Python ва Aiogram таҳия шудааст. Лоиҳа барои автоматизатсия кардани фурӯши маҳсулот, 
идоракунии фармоишҳо ва осон намудани хизматрасонӣ ба мизоҷон пешбинӣ шудааст.
```
✨ Имкониятҳо
📦 Намоиши маҳсулот аз рӯи категорияҳо
🛒 Илова кардани маҳсулот ба сабад
📋 Дидани маълумоти пурраи маҳсулот
💳 Сабти фармоиш
👤 Панели корбар
🔐 Панели администратор
➕ Илова, таҳрир ва нест кардани маҳсулот
📢 Ирсоли паём ба ҳамаи истифодабарандагон
🗂️ Идоракунии категорияҳо
📊 Нигоҳдории маълумот дар пойгоҳи додаҳо
```

```
🛠️ Технологияҳои истифодашуда
Python 3.x
Aiogram
SQLite / PostgreSQL
SQLAlchemy
AsyncIO
```

python app.py

python main.py
